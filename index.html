倉頡新星

倉頡新星是倉頡輸入法教學軟體 倉頡之星 的仿製版， 讓你從不會倉頡，到看到字都會想用倉頡字根拼出來 XD

為什麼用倉頡？

嘸蝦米要錢，而且用英文拆中文實在很鳥。 大易 40 鍵太多，行列圈圈太小。 唯有倉頡，一路走來，四平八穩。 什麼你說注音？那能吃嗎？

倉頡教學

看倉頡如何把一個字拆成 字首 、 字根 ， 各字根如何變化組合，成成千上萬個中文字。

鍵位練習

打字看著鍵盤，傷脊椎； 打字看著螢幕，傷眼睛。 學盲打，讓你抬頭挺胸； 學倉頡，讓你一望無際。

拆字練習

來實際個字看看吧！ 4800 個常用字，依難易度分為 31 個字庫。 但其實不用打玩，拆完一二個字庫，就可以上了。 日常的使用就沒問題了。

有關倉頡新星

本軟體使用 HTML + CSS + JavaScript 寫成， 如果出現 bug ，請先更新瀏覽器看看。 除了有線上版，本軟體也有離線版。 （當然也是用 HTML + CSS + JavaScript 寫的， 也是要用瀏覽器開，就相當於離線的網頁。） 兩者功能相同，只是線上版可能會有更新， 而離線版要更新就要自己重新下載新的版本。
